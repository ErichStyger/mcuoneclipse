Component GenericI2C
Generic high level I2C Driver on top of the I2C hardware driver
Component Level: Low
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • SelectSlave - Selects the slave device on the bus. Method might use a semaphore to protect bus access.

      ANSIC prototype: byte SelectSlave(byte i2cAddr)
    • i2cAddr:byte - I2C Address of device
    • Return value:byte - Error code

  • UnselectSlave - Unselects the device. Method will release a used a semaphore.

      ANSIC prototype: byte UnselectSlave(void)
    • Return value:byte - Error code

  • RequestBus - Starts a critical section for accessing the bus.

      ANSIC prototype: void RequestBus(void)

  • ReleaseBus - Finishes a critical section for accessing the bus.

      ANSIC prototype: void ReleaseBus(void)

  • ReadBlock - Read from the device a block.

      ANSIC prototype: byte ReadBlock(void* data, word dataSize, ComponentName_EnumSendFlags flags)
    • data:void* - Read buffer
    • dataSize:word - Size of read buffer
    • flags:byte - flags for the transaction
    • Return value:byte - Error code

  • WriteBlock - Write a block to the device.

      ANSIC prototype: byte WriteBlock(void* data, word dataSize, ComponentName_EnumSendFlags flags)
    • data:void* - Data write buffer
    • dataSize:word - no hint
    • flags:byte - flags for the transaction
    • Return value:byte - Error code

  • ReadAddress - Read from the device. This writes (S+i2cAddr+0), (memAddr), (Sr+i2cAddr+1), (data)...(data+P)

      ANSIC prototype: byte ReadAddress(byte i2cAddr, byte *memAddr, byte memAddrSize, byte *data, word dataSize)
    • i2cAddr:byte - I2C Address of device
    • memAddr: Pointer to byte - Pointer to device memory address
    • memAddrSize:byte - number of address bytes
    • data: Pointer to byte - Pointer to read buffer
    • dataSize:word - Size of read buffer
    • Return value:byte - Error code

  • WriteAddress - Write to the device: (S+i2cAddr+0), (memAddr), (data)...(data+P)

      ANSIC prototype: byte WriteAddress(byte i2cAddr, byte *memAddr, byte memAddrSize, byte *data, word dataSize)
    • i2cAddr:byte - I2C address of device
    • memAddr: Pointer to byte - Pointer to device memory address
    • memAddrSize:byte - number of address bytes
    • data: Pointer to byte - Pointer to data write buffer
    • dataSize:word - Size of data buffer in bytes
    • Return value:byte - Error code

  • ReadByteAddress8 - Read a byte from the device using an 8bit memory address. This writes (S+i2cAddr+0), (memAddr), (Sr+i2cAddr+1), (data)...(data+P)

      ANSIC prototype: byte ReadByteAddress8(byte i2cAddr, byte memAddr, byte *data)
    • i2cAddr:byte - I2C Address of device
    • memAddr:byte - Device memory address
    • data: Pointer to byte - Pointer to read buffer (single byte)
    • Return value:byte - Error code

  • WriteByteAddress8 - Write a byte to the device using an 8bit memory address: (S+i2cAddr+0), (memAddr), (data)...(data+P)

      ANSIC prototype: byte WriteByteAddress8(byte i2cAddr, byte memAddr, byte data)
    • i2cAddr:byte - I2C address of device
    • memAddr:byte - Device memory address
    • data:byte - Data value
    • Return value:byte - Error code

  • ProbeACK - Accesses the bus to check if the device responds with an ACK (ACK polling).

      ANSIC prototype: byte ProbeACK(void* data, word dataSize, ComponentName_EnumSendFlags flags, word WaitTimeUS)
    • data:void* - Data write buffer
    • dataSize:word - no hint
    • flags:byte - flags for the transaction
    • WaitTimeUS:word - Waiting time in microseconds to wait for the ACK on the bus.
    • Return value:byte - Error code

  • GetSemaphore - Returns the currently allocated semaphore.

      ANSIC prototype: void* GetSemaphore(void)
    • Return value:void* - Error code

  • Deinit - Deinitializes the driver.

      ANSIC prototype: void Deinit(void)

  • ScanDevice - Checks if a device responds on the bus with an ACK.

      ANSIC prototype: byte ScanDevice(byte i2cAddr)
    • i2cAddr:byte - 7bit I2C device address
    • Return value:byte - Error code

  • Init - Initializes the driver.

      ANSIC prototype: void Init(void)


Note: Some methods can be implemented as macros.
