Component LCDHTA
Character LCD Display
Component Level: High
Category: CPU External Devices-Display
This bean implements a flexible LCD display driver.
Supported displays are the ones compatible with the Hitachi HD44780 or similar.
Additional information on how to use such a LCD controller is explained in www.freescale.com/files/microcontrollers/doc/app_note/AN1745.pdf
The driver supports following hardware connection methods:
- 4bit data/control interface
- 8bit data/control interface
- optional display select (for having multiple displays on the bus)

Example Board Configuration

Below a list of some board configuration:

Freescale MC13213 NCB Board (ZigBee):
- 2 Lines with 16 characters each
- Device: Generic
- LCD_Enable Signal: B0
- Check Busy Flag: yes - RS: B1
- RW: B3
- EN: B2
- Data bus: 4bit: B4..B7

Motorola 9S08GB60 ABS512 Board:
Configuration with original PWB16117L-CEM display (5V):
- 1 Line with 16 characters
- Device: Generic
- LCD_Enable Signal: disabled
- Check Busy Flag: no - RS: F1
- RW: F0
- EN: F2
- Data bus: 4bit: A0..A3

Configuration with DisplayTec 162C display (5V):
- 2 Lines with 16 characters
- Device: DisplayTec 162C
- LCD_Enable Signal: disabled
- RS: F1
- RW: F0
- EN: F2
- Data bus: 4bit: A0..A3

Infotronic class at the University of Lucerne:
In the class a blue 3.3V LCD is used:
- 2 Lines with 16 characters each
- Device: Generic
- LCD_Enable Signal: disabled
- Check Busy Flag: yes - RS: B0
- RW: B1
- EN: B2
- Data bus: 8bit: C0..C7 or 4bit: C4..C7
