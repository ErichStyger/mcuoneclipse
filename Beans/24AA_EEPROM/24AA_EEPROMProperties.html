Component 24AA_EEPROM
Microchip 24AA Serial EEPROM
Component Level: Low
Category: CPU External Devices-Memory
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Device - Specifies the device type used.

    There are 5 options:
    • 8: 1 KByte: 24LC08
    • 32: 4 KByte: 24LC32
    • 256: 32 KByte: 24AA256/24LC256/24FC256
    • 512: 64 KByte: 24AA512/24LC512/24FC512
    • 1025: 128 KByte: 24AA1025/24LC1025/24FC1025

  • Initial Device Address - The address must correspond to the logic levels on the corresponding address pins of the device which form the I2C device address. You can change the address at runtime using SetAddress().
  • Block buffer size - Buffer size for block writing. The smaller the value, the less stack is used. This limits as well the maximum of block size which can be written.
  • Acknowledge Polling - If enabled, the driver will perform Acknowledge Polling as outlined in the data sheet of the device. Otherwise it is in the responsibility of the application to perform this.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Page Write Time (ms) - Specifies the typical page write time, as per data sheet. This is the time needed for the device to write a page. This time will be used for the acknowledge polling as initial waiting time.
      Typical values:
      24xx8: 3 ms
      24xx512: 5 ms
    • Wait - Interface to waiting routines. Driver will use RTOS waiting routine if RTOS is present.
  • Connection - Connection settings.
    • I2C - Interface to I2C.
    • Write Protection Pin - If not hardwired, the WP (Write Protection) pin allows to prevent writing to the memory.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • WP - Write protect pin, high active. If connected to Vcc/High, then write operations are inhibited, but read operations are not affected.
