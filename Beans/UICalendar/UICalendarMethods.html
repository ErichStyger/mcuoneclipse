Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • ChangeCalendarDay - Increases or decreases the calendar day for a calendar day item. The method cares as well about leap years.

      ANSIC prototype: void ChangeCalendarDay(ComponentName_Window *window, ComponentName_CalendarDay *calendarDay, signed char changeDays, int16_t changeMinutes)
    • window: Pointer to ComponentName_Window - Pointer to window
    • calendarDay: Pointer to ComponentName_CalendarDay - Pointer to calendar day item
    • changeDays:signed char - number of days to increment. Note that the argument is positive for the future and negative to move the date back
    • changeMinutes:int16_t - Number of minutes to change the start time. Positive to change it to the future, and negative to change it to the back.

  • CreateCalendarDay - Creates a new calendar day element

      ANSIC prototype: byte CreateCalendarDay(ComponentName_Window *window, ComponentName_CalendarDay *calendarDay, ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim width, ComponentName_PixelDim height, ComponentName_CalendarDateTime *start)
    • window: Pointer to ComponentName_Window - Pointer to
    • calendarDay: Pointer to ComponentName_CalendarDay - Pointer to calendar day element
    • x:ComponentName_PixelDim - x position
    • y:ComponentName_PixelDim - y position
    • width:ComponentName_PixelDim - width of element
    • height:ComponentName_PixelDim - height of element
    • start: Pointer to ComponentName_CalendarDateTime - Pointer to start date/time for calendar
    • Return value:byte - Error code

  • GetCalendarDayResolution - Returns the resolution of calender items (e.g. 30 for 30 minutes resolution)

      ANSIC prototype: byte GetCalendarDayResolution(void)
    • Return value:byte - Error code


Note: Some methods can be implemented as macros.
