Component S65Sharp
Display driver for the Siemens S65 display (using Sharp LS020xxx controller)
Component Level: Low
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Orientation - Orientation of the display.

    There are 4 options:
    • landscape: Landscape format
    • landscape180: Landscape, rotated by 180 degree
    • portrait: Portrait, landscape rotated right by 90 degree
    • portrait180: Portrait, landscape rotated left by 90 degree

  • Inverted display - If display colors shall be inverted
  • Initialize on Init - If the driver shall be initialized during Init
  • Properites -
    • Width - Width of the display in pixels.
    • Bytes in rows - If bytes are organized in rows or columns.
    • Bytes in x direction - If bytes are increasing in x direction (from left to right). Otherwise bytes are increasing in y direction (from top to down).
    • MSB first - If the Most Significant bit/pixel comes first in the byte.
    • Bits per pixel - Number of bits per pixel. 1 for B/W displays.
    • Window capability - If the display is able to use a local window on the device to write data.
    • Display Memory Write - If we can directly write into the display memory
    • Height - The height of the display in pixels.
  • HW - Group controlling the hardware settings.
    • RES - Reset signal pin
    • SCE - SPI Chip Enable/Select signal (low active)
    • D_C - Data/Command mode select pin.
    • SW SPI - Group to control the Software SPI.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • SW SPI - Software SPI interface
    • HW SPI - This sets up a Hardware SPI communication
      The following items are available only if the group is enabled (the value is "Enabled"):
      • HW SPI - Hardware SPI interface
  • System - Group controlling the system settings.
    • Wait - Interface for busy waiting.
