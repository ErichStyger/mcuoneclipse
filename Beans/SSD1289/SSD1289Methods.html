Component SSD1289
Display driver for the Solomon Systech SSD1289 display
Component Level: Low
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetWidth - Returns the width of the display in pixels (in x direction)

      ANSIC prototype: ComponentName_PixelDim GetWidth(void)
    • Return value:ComponentName_PixelDim - Error code

  • GetHeight - Returns the height of the display in pixels (in y direction)

      ANSIC prototype: ComponentName_PixelDim GetHeight(void)
    • Return value:ComponentName_PixelDim - Error code

  • GetLongerSide - Returns the size of the longer side of the display

      ANSIC prototype: ComponentName_PixelDim GetLongerSide(void)
    • Return value:ComponentName_PixelDim - Error code

  • GetShorterSide - Returns the size of the shorter side of the display

      ANSIC prototype: ComponentName_PixelDim GetShorterSide(void)
    • Return value:ComponentName_PixelDim - Error code

  • SetDisplayOrientation - Sets the display orienation. If you enable this method, then the orientation of the display can be changed at runtime. However, this requires additional ressources.

      ANSIC prototype: void SetDisplayOrientation(ComponentName_DisplayOrientation newOrientation)
    • newOrientation:ComponentName_DisplayOrientation - new orientation to be used

  • GetDisplayOrientation - Returns the current display orientation

      ANSIC prototype: ComponentName_DisplayOrientation GetDisplayOrientation(void)
    • Return value:ComponentName_DisplayOrientation - current display orientation

  • ReadDataWord - Writes a single word to the bus

      ANSIC prototype: word ReadDataWord(void)
    • Return value:word - data word from the display

  • WriteDataWord - Writes a single word to the bus

      ANSIC prototype: void WriteDataWord(word data)
    • data:word - Word to send

  • WriteCommandWord - Sends a word as command

      ANSIC prototype: void WriteCommandWord(word cmd)
    • cmd:word - the command to be sent

  • WriteCommandData - Sends multiple words to the display as commands + data sequence

      ANSIC prototype: void WriteCommandData(const word* data, byte nofWords)
    • data: Pointer to word - Pointer to the data to write (nofWords of words)
    • nofWords:byte - number of words to send.

  • OpenWindow - Opens a window inside the display we want to write to.

      ANSIC prototype: void OpenWindow(ComponentName_PixelDim x0, ComponentName_PixelDim y0, ComponentName_PixelDim x1, ComponentName_PixelDim y1)
    • x0:ComponentName_PixelDim - no hint
    • y0:ComponentName_PixelDim - no hint
    • x1:ComponentName_PixelDim - no hint
    • y1:ComponentName_PixelDim - no hint

  • CloseWindow - Closes a window previously opened with OpenWindow()

      ANSIC prototype: void CloseWindow(void)

  • Clear - Clears the whole display memory.

      ANSIC prototype: void Clear(void)

  • UpdateFull - Updates the whole display. This is only a stub for this display as we are using windowing.

      ANSIC prototype: void UpdateFull(void)

  • UpdateRegion - Updates a region of the display. This is only a stub for this display as we are using windowing.

      ANSIC prototype: void UpdateRegion(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim w, ComponentName_PixelDim h)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - y coordinate
    • w:ComponentName_PixelDim - width of the region
    • h:ComponentName_PixelDim - Height of the region

  • Init - Display driver initialization

      ANSIC prototype: void Init(void)

  • InitCommChannel - Method to initialize communication to the LCD. Needed if the bus to the LCD is shared with other components and settings are different.

      ANSIC prototype: void InitCommChannel(void)

  • GetLCD - Method to be called for mutual exclusive access to the LCD

      ANSIC prototype: void GetLCD(void)

  • GiveLCD - Method to be called for mutual exclusive access to the LCD

      ANSIC prototype: void GiveLCD(void)


Note: Some methods can be implemented as macros.
