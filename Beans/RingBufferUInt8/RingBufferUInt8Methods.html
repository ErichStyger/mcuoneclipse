Component RingBufferUInt8
Implements a byte ring buffer
Component Level: High
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Clear - Clear (empty) the ring buffer.

      ANSIC prototype: void Clear(void)

  • Put - Puts a new element into the buffer

      ANSIC prototype: byte Put(byte elem)
    • elem:byte - New element to be put into the buffer
    • Return value:byte - Error code

  • Get - Removes an element from the buffer

      ANSIC prototype: byte Get(byte *elemP)
    • elemP: Pointer to byte - Pointer to where to store the received element
    • Return value:byte - Error code

  • NofElements - Returns the actual number of elements in the buffer.

      ANSIC prototype: byte NofElements(void)
    • Return value:byte - Number of elements in the buffer.

  • NofFreeElements - Returns the actual number of free elements/space in the buffer.

      ANSIC prototype: byte NofFreeElements(void)
    • Return value:byte - Number of elements in the buffer.

  • Init - Initializes the data structure

      ANSIC prototype: void Init(void)


Note: Some methods can be implemented as macros.
