FreeRTOS Trace
Component Level: High

Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Reserve - Reserves the resource

      ANSIC prototype: void Reserve(void)

  • Unreserve - Unreserves the resource

      ANSIC prototype: void Unreserve(void)

  • Select - Select the communication channel

      ANSIC prototype: byte Select(void)
    • Return value:byte - Error code

  • Unselect - Unselect the communication channel

      ANSIC prototype: byte Unselect(void)
    • Return value:byte - Error code

  • SendByte - Send a Byte to the Probe

      ANSIC prototype: byte SendByte(byte val)
    • val:byte - The byte to be sent
    • Return value:byte - Error code

  • Enable - Enable the trace to the Probe

      ANSIC prototype: void Enable(void)

  • Disable - Disable the trace to the Probe must called before the FreeRTOS method task create is called

      ANSIC prototype: void Disable(void)

  • Init - Init the Component

      ANSIC prototype: void Init(void)


Note: Some methods can be implemented as macros.
