Component RNet
RNet (Radio and Remote Network) Stack
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Transceiver - Transceiver configuration items
    • Transceiver Type - Type of Transeiver

      There are 2 options:
      • nRF24L01+: Nordic Semiconductor nRF24L01+
      • MC13201: Freescale Semiconductor MC13201/MC13213

    • nRF24L01+ - nRF24L01+ settings
      The following items are available only if the group is enabled (the value is "Enabled"):
      • nRF24L01+ - Link to nRF24L01+ transceiver
      • Radio Channel - Default channel
      • Data Rate - Default data rate of transceiver

        There are 3 options:
        • 250 kBit: 255 kBit
        • 1000 kBit: 1000 kBit
        • 2000 kBit: 2000 kBit

      • Payload Size - Maximum number of bytes in a single message. Cannot exceed physical transceiver message size, and below the minimal size including overhead.
    • SMAC - MC13202/MC13192 transceiver stack
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Radio Channel - Default radio transceiver channel, in the range 0..15
      • Output Power - Default output power, in the range of 0..15 with 15 the maximum power.
      • SMAC - Interface to the SMAC stack
  • Network - Network settings. These settings need to be the same for all nodes in a network.
    • Short Address Size - Size of the node network address size. Note that this size needs to be the same for all nodes in a network.

      There are 2 options:
      • 8 Bits: 8bit network node address
      • 16 Bits: 16bit network node address

    • Use ACK - If enabled, the NWK layer will respond to ACK requests (only for messages which requests this).
  • Queues - Settings for the internal message Queues
    • Rx Message Queue Size - Number of possible items in the RX queue
    • Tx Message Queue Size - Number of possible items in the TX queue
    • Message Queue Blocking Time (ms) - Blocking time for putting messages into the queue. This specifies the time function will wait until there is space in the queue.
  • Send Retry Count - Number of message send retry if it failed. Set it to zero to disable retry.
  • System - Configuration of the System
    • Utility - Interface to Utility functions
    • RTOS - Operating System
    • Shell - If shell support is enabled. This enables/disables the method ParseCommand().
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Remote StdIO - Enables stdio handling
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Queue length - Size of queue length (RSTDIO_QUEUE_LENGTH)
        • Queue Timeout (ms) - Timeout for stdio queues (RSTDIO_QUEUE_TIMEOUT_MS)
      • Shell - Interface to Shell
