Component MMA8451Q
Driver for the Freescale MMA8451Q digital I2C accelerometer.
Component Level: Low
Category: CPU External Devices-Sensor
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component Name - Name of the component.
  • Slave Address - Specifies the I2C slave address of the accelerometer.

    There are 2 options:
    • 1D: I2C Device Slave Address, with SA0 (pin7) set to logical HIGH level.
    • 1C: I2C Device Slave Address, with SA0 (pin7) set to logical LOW level.

  • I2C - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
