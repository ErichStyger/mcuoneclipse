Component minIni
minIni is a programmer’s library to read and write 'ini' files in embedded systems.
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • minIni Version - Identifies the minIni version
  • FatFS - FatFS configuration items
    • FatFS - Link to FatFS (FAT File System) component.
    • Buffer Size - Configures the macro INI_BUFFERSIZE which configures the maximum line length adn maxium path length.
