Component AT25HP512
Driver for SPI EEPROM AT25HP512
Component Level: Low
Category: CPU External Devices-Memory
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • WriteBlock - Writes a block of memory to the EEPROM. Note that the EEPROM only allows 128 byte page write operation. As such writing just a byte will require to read a full page and to write a full page.

      ANSIC prototype: byte WriteBlock(EEPROM_Address addr, byte *data, word size)
    • addr:EEPROM_Address - no hint
    • data: Pointer to byte - no hint
    • size:word - no hint
    • Return Value:byte - no hint

  • ReadBlock - Reads a block of memory from the EEPROM.

      ANSIC prototype: byte ReadBlock(EEPROM_Address addr, byte *data, word size)
    • addr:EEPROM_Address - no hint
    • data: Pointer to byte - no hint
    • size:word - no hint
    • Return Value:byte - no hint

  • ReadByte - Reads a single byte from the EEPROM.

      ANSIC prototype: byte ReadByte(EEPROM_Address addr, byte *data)
    • addr:EEPROM_Address - no hint
    • data: Pointer to byte - no hint
    • Return Value:byte - no hint


Note: Some methods can be implemented as macros.
