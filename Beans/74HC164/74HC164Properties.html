Component 74HC164
74HC164 Shift Register
Component Level: Low
Category: CPU External Devices-Port I/O
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Hardware -
    • CLK - Clock pin (low to high active)
    • A - A pin
    • CLR - The CLR signal is low active. If you do not use it, set it to a logical high level.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • CLR - Clear pin (low active)
    • B - The B pin can be used as 'active high enable'
      The following items are available only if the group is enabled (the value is "Enabled"):
      • B - B pin
  • System -
    • Wait - BusyWait Interface
