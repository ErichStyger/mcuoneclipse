Component FXOS8700CQ
Driver for the Freescale FXOS8700CQ digital I2C accelerometer and magnetometer.
Component Level: Low
Category: CPU External Devices-Sensor
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component Name - Name of the component.
  • Slave Address - Specifies the I2C slave address of the accelerometer.

    There are 4 options:
    • 1E: I2C Device Slave Address, with SA1 set to LOW and SA0 set to LOW.
    • 1D: I2C Device Slave Address, with SA1 set to LOW and SA0 set to HIGH.
    • 1C: I2C Device Slave Address, with SA1 set to HIGH and SA0 set to LOW.
    • 1F: I2C Device Slave Address, with SA1 set to HIGH and SA0 set to HIGH.

  • I2C Bus - Interface to I2C bus connected to the accelerometer.
  • Wait - Interface to waiting routines
  • Temperature offset - The DIE_TEMP register is not factory trimmed, so returns around 0°C for about 20°C. With this offset can be trimmed.
  • Constant Offsets - If enabled, constant offsets are added to the accelerometer values.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • X offset - Offset to be added to X acceleration sensor value
    • Y offset - Offset to be added to Y acceleration sensor value
    • Z offset - Offset to be added to Z acceleration sensor value
  • Shell - If shell support is enabled. This enables/disables the method ParseCommand().
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Shell - Interface to shell
    • Utility - Interface to Utility functions
