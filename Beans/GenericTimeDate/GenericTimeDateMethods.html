Component GenericTimeDate
Generic Software Time and Date component
Component Level: High
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • AddTick - Needs to be called periodically by the application to increase the time tick count.

      ANSIC prototype: void AddTick(void)

  • SetTime - This method sets a new actual time.

      ANSIC prototype: uint8_t SetTime(uint8_t Hour, uint8_t Min, uint8_t Sec, uint8_t Sec100)
    • Hour:uint8_t - Hours (0 - 23)
    • Min:uint8_t - Minutes (0 - 59)
    • Sec:uint8_t - Seconds (0 - 59)
    • Sec100:uint8_t - Hundredth of seconds (0 - 99)
    • Return value:uint8_t - Error code

  • GetTime - This method returns current time.

      ANSIC prototype: uint8_t GetTime(TIMEREC *Time)
    • Time: Pointer to TIMEREC - Pointer to the structure TIMEREC. It contains actual number of hours, minutes, seconds and hundredth of seconds.
    • Return value:uint8_t - Error code

  • SetDate - This method sets a new actual date.

      ANSIC prototype: uint8_t SetDate(uint16_t Year, uint8_t Month, uint8_t Day)
    • Year:uint16_t - Years (16-bit unsigned integer)
    • Month:uint8_t - Months (8-bit unsigned integer)
    • Day:uint8_t - Days (8-bit unsigned integer)
    • Return value:uint8_t - Error code

  • GetDate - This method returns current date.

      ANSIC prototype: uint8_t GetDate(DATEREC *Date)
    • Date: Pointer to DATEREC - Pointer to DATEREC
    • Return value:uint8_t - Error code

  • ParseCommand - Shell Command Line parser

      ANSIC prototype: uint8_t ParseCommand(const unsigned char *cmd, bool *handled, const %@Shell@'ModuleName'%.StdIOType *io)
    • cmd: Pointer to uint8_t - Pointer to command line
    • handled: Pointer to bool - Pointer to variable which tells if the command has been handled or not
    • io: Pointer to uint8_t - Pointer to I/O structure
    • Return value:uint8_t - Error code

  • Init - Initialization method

      ANSIC prototype: void Init(void)

  • DeInit - Deinitializes the driver.

      ANSIC prototype: void DeInit(void)


Note: Some methods can be implemented as macros.
