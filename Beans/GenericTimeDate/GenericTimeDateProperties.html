Component GenericTimeDate
Generic Software Time and Date component
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Critical Section - Interface to critical section handling
  • Tick Time (ms) - Tick timer time in milli seconds. This is the frequency at which AddTick() is called.
  • RTOS - If enabled, it is using automatically the period of the RTOS tick counter. Still you need to call the AddTick() method from the RTOS tick counter event callback.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • RTOS - Interface to RTOS. You need to call from the tick counter the AddTick() method of this component.
  • Shell - Optional shell command line interface
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Utility - Interface to utility
    • Shell - Interface to shell
  • Initialization - If enabled, the component will use initialization values in Init() method.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Init in startup - If the Init() routine shall be called from PE_low_level_init()
    • Date - Initial Date
    • Time - Initial Time
