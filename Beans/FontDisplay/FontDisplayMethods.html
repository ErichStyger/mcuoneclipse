Component FontDisplay
Graphical font driver
Component Level: High
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetFontHeight - returns the height (in pixels) of a font.

      ANSIC prototype: void GetFontHeight(ComponentName_Font *font, ComponentName_PixelDim *charHeight, ComponentName_PixelDim *totalHeight)
    • font: Pointer to ComponentName_Font - Pointer to font information
    • charHeight: Pointer to ComponentName_PixelDim - Pointer to height of the character in pixels.
    • totalHeight: Pointer to ComponentName_PixelDim - Pointer to the total height of the character, including to the space below the character.

  • GetStringHeight - returns the height (in pixels) of a font string. For multiple line strings (separated by newline) or for strings with \r it returns the longest list.

      ANSIC prototype: ComponentName_PixelDim GetStringHeight(byte *str, ComponentName_Font *font, ComponentName_PixelDim *lastLineSpace)
    • str: Pointer to byte - Pointer to the string.
    • font: Pointer to ComponentName_Font - Pointer to font information
    • lastLineSpace: Pointer to ComponentName_PixelDim - Pointer to the height of the last line space, useful if you combine multiple strings. The pointer may be NULL.
    • Return value:ComponentName_PixelDim - Error code

  • GetCharWidth - returns the width (in pixels) of a font character.

      ANSIC prototype: void GetCharWidth(byte ch, ComponentName_PixelDim *charWidth, ComponentName_PixelDim *totalWidth, ComponentName_Font *font)
    • ch:byte - character for which to calculate the width
    • charWidth: Pointer to ComponentName_PixelDim - Pointer to width of the character in pixels (this is only for the character, without the space after the character)
    • totalWidth: Pointer to ComponentName_PixelDim - Pointer to the total width of the character, including to the space after the character.
    • font: Pointer to ComponentName_Font - Pointer to font information

  • GetStringWidth - returns the width (in pixels) of a font string. For multiple line strings (separated by \
    ) or for strings with \r it returns the longest string length.

      ANSIC prototype: ComponentName_PixelDim GetStringWidth(byte *str, ComponentName_Font *font, ComponentName_PixelDim *lastCharSpace)
    • str: Pointer to byte - Pointer to the string.
    • font: Pointer to ComponentName_Font - Pointer to font information
    • lastCharSpace: Pointer to ComponentName_PixelDim - Pointer to the width of the last character space, useful if you combine multiple strings. The pointer may be NULL.
    • Return value:ComponentName_PixelDim - Error code

  • WriteString - Function to write a string or character at the startpoint. Set the startpoint with the function setCursor.

      ANSIC prototype: void WriteString(byte *str, ComponentName_PixelColor color, ComponentName_PixelDim *xCursor, ComponentName_PixelDim *yCursor, ComponentName_Font *font)
    • str: Pointer to byte - Pointer to the character string
    • color:ComponentName_PixelColor - Foreground color to be used (for the character pixels)
    • xCursor: Pointer to ComponentName_PixelDim - Pointer to x position of first character (upper left corner). On return this will contain the next y position.
    • yCursor: Pointer to ComponentName_PixelDim - Pointer to y position of character (upper left corner). On return this will contain the next y position.
    • font: Pointer to ComponentName_Font - Pointer to font information

  • WriteChar - Function to write a character at the start point. Set the start point with the function setCursor.

      ANSIC prototype: void WriteChar(byte ch, ComponentName_PixelColor color, ComponentName_PixelDim *xCursor, ComponentName_PixelDim *yCursor, ComponentName_Font *font)
    • ch:byte - the character to print
    • color:ComponentName_PixelColor - Foreground color to be used (for the character pixels)
    • xCursor: Pointer to ComponentName_PixelDim - Pointer to x position of character (upper left corner). On return this will contain the next x position.
    • yCursor: Pointer to ComponentName_PixelDim - Pointer to y position of character (upper left corner). On return this will contain the next y position.
    • font: Pointer to ComponentName_Font - Pointer to font information


Note: Some methods can be implemented as macros.
