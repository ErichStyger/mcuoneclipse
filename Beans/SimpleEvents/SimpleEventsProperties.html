Component SimpleEvents
Simple Event Handling
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Initialize on Init - If driver data shall be initialized during init
  • Event Name List - List of events. The event names will be published in the header file.
  • Power Save - if enabled, then in case there are no events, the driver will enter a low power/sleep mode. This requires that the CPU offers this.
  • Low Power - If there is a need for special handling in low power mode
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Low Power - Configures the low power handling
