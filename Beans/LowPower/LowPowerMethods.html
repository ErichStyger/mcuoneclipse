Component LowPower
Provides support for low power modes
Component Level: Low
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • IsRecoveryThroughReset - Returns true if we went through reset because of recovery from a low power mode.

      ANSIC prototype: bool IsRecoveryThroughReset(void)
    • Return value:bool - if we went through reset sequence because of recovery

  • SetRecoveryThroughReset - Called in case we were going through reset because of low power recovery.

      ANSIC prototype: void SetRecoveryThroughReset(bool value)
    • value:bool - the value to be set

  • InitializeRAMinInit - no hint

      ANSIC prototype: bool InitializeRAMinInit(void)
    • Return value:bool - if we need to initialize the RAM in Init

  • EntryPoint - no hint

      ANSIC prototype: void EntryPoint(void)


Note: Some methods can be implemented as macros.
