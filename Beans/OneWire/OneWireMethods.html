  • Receive - Programs a read operation after the master send all in output buffer. Don't use a SendReset while the data is coming.

      ANSIC prototype: bool Receive(uint8_t numOfBytes,uint8_t count)
    • numOfBytes:uint8_t -
    • count:uint8_t - Number of bytes to receive from slave.
    • Return value:bool - Returns FALSE if the program buffer is full or if count is greater than the free space of input buffer-

      ANSIC prototype: uint8_t Count(void)

      ANSIC prototype: bool ProgramEvent(uint16_t key)
    • key:uint16_t - Key to identificate the event, applies only if OProgramEvent is enabled. (Valid range 0 - 31)
    • Return value:bool - Returns false if the program buffer is full.

  • Waitms - Programs a pause between instruccions.

      ANSIC prototype: bool Waitms(uint8_t key,uint16_t time_ms)
    • key:uint8_t - Key to identificate the period end, applies only if OnWaitEnd evet is enabled. (Valid range 0 - 31)
    • time_ms:uint16_t - Value of time to wait.
    • Return value:bool - Returns false if the program buffer or the time buffer is full.
