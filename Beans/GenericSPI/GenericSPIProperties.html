Component GenericSPI
Generic SPI implementation
Component Level: Low
Category: CPU Internal Peripherals-Communication
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • SPI - Enables or disables the interface
    The following items are available only if the group is enabled (the value is "Enabled"):
    • SPI Channel -
  • SPI Register -
    • Status register - SPI Status register (8bit)
    • Data register - SPI Data register (8bit)
    • Read Buffer Full Flag Bit - SPI Read Buffer Full Flag bit. The driver will use this flag (check if cleared) to see if the read buffer is full and we can read a byte.
    • Write Buffer Empty Flag Bit - SPI Write Buffer Emtpy Flag bit. The driver will use this flag (check if cleared) to see if the write buffer is empty and we can write a new byte to the bus.
