Component LED
Driver for a LED
Component Level: Low
Category: CPU External Devices-Display
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Turned On with initialization - If the LED shall be on or off after initialization.
  • HW Interface - This group controls how the componet is using the hardware
    • High pin value means ON - If a high value means 'on'. If the anode side of the LED is connected to the port (otherwise it is the kathode). If the LED anode is on the port side, then a logical one on the port turns the LED on. If the cathode is on the port side, then a low level will turn it on.
    • On/Off - If enabled, implements a normal on and off pin
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Pin - Interface to the LED pin
    • PWM - If enabled, a PWM signal is used to implement a dimmable LED.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • inverted - If PWM is inverted
      • PWM - Interface to PWM.
  • Shell - If shell support is enabled.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Shell - Interface to shell
    • Utility - Interface to Utility functions
