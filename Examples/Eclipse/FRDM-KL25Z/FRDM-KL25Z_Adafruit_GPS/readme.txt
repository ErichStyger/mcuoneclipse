This project implements a GPS data logger based on the Adafruit GPS Data Logger Shield.
As toolchain Eclipse Kepler with GNU ARM Embedded and Processor Expert is used.

See
http://mcuoneclipse.com/2014/05/31/tutorial-freedom-board-with-adafruit-ultimate-gps-data-logger-shield/


Hint: How to import GPS data into Google Earth:
http://www.gpsvisualizer.com/map_input?form=googleearth
and convert a CSV text file like
name,latitude,longitude
-, +47° 00.2806', +008° 35.1101'
-, +47° 00.2907', +008° 35.1101'
-, +47° 00.3008', +008° 35.1101'
